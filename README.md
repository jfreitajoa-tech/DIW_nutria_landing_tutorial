# NutrIA · Landing Videotutorial

Este proyecto corresponde a la actividad de grabación y presentación del prototipo **NutrIA**.

Se trata de una landing page desarrollada en HTML y Tailwind CSS donde se integra el videotutorial explicativo del prototipo web creado previamente.

---

## Sobre el proyecto

La landing tiene como objetivo presentar de forma clara y visual el funcionamiento del prototipo NutrIA mediante un vídeo demostrativo.

En la página se incluye:

- Encabezado con identidad visual del proyecto  
- Breve introducción explicativa  
- Videotutorial integrado con controles  
- Subtítulos en español (formato `.vtt`)  
- Información sobre las pantallas mostradas  
- Enlace directo al prototipo publicado en GitHub Pages  

---

## Videotutorial

El vídeo muestra:

- Pantalla principal (home)  
- Listado de recetas  
- Vista detalle de receta  
- Página de contacto  
- Adaptación responsive en diferentes tamaños de pantalla  

Incluye subtítulos para mejorar la accesibilidad.

---

## Enfoque

El diseño sigue un enfoque:

- Mobile first  
- Responsive  
- Navegación clara  
- Organización visual del contenido
